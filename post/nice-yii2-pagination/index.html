Удобная пагинация в yii2

May 23, 2017

2 minute read

  • Twitter

  • Twitter

  • Google+

  • Facebook

  • LinkedIn

  • Email

Добиваемся пагинации в yii2 такого вида:

В чем смысл:

  • Всегда видно 7 ближайших страниц.
  • Всегда можно перейти на первую или последнюю страницу.
  • Ссылка “на первую страницу” появляется только при необходимости: если первая страница еще не показывается среди ближайших 7-ми.
  • Ссылка “на предыдущую страницу” появляется только тогда, когда есть ссылка на первую страницу.
  • Аналогично со ссылками “вперед”.

Настройки для yii\widgets\LinkPager:

$pages = ... ; // это должен быть yii\data\Pagination
$linkPagerConfig = [
    'prevPageLabel' => ($pages->pageCount <= 7) || ($pages->page <= 3) ? false : '<',
    'firstPageLabel' => ($pages->pageCount <= 7) || ($pages->page <= 3) ? false : true,
    'nextPageLabel' => ($pages->pageCount <= 7) || ($pages->page >= $pages->pageCount - 1 - 3) ? false : '>',
    'lastPageLabel' => ($pages->pageCount <= 7) || ($pages->page >= $pages->pageCount - 1 - 3) ? false : true,
    'maxButtonCount' => 7
];

Если LinkPager используется внутри GridView или другим образом через DataProvider, то сначала нужно проинициализировать DataProvider, чтобы он проинициализировал пагинацию.

$dataProvider->prepare(); // инициализируем, чтобы проинициализировалась пагинация
$pages = $dataProvider->pagination;

Вариант для twig:

{% set total_pages = pages.pageCount - 1 %}
{% set pagination = link_pager_widget({
    pagination: pages,
    prevPageLabel: (pages.pageCount <= 7) or (pages.page <= 3)  ? false : '<',
    firstPageLabel: (pages.pageCount <= 7) or (pages.page <= 3) ? false : true,
    nextPageLabel: (pages.pageCount <= 7) or (pages.page >= total_pages - 3) ? false : '>',
    lastPageLabel: (pages.pageCount <= 7) or (pages.page >= total_pages - 3) ? false : true,
    maxButtonCount: 7,
}) %}
{{ pagination|raw }}
