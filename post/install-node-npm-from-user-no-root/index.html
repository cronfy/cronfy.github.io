Установка node/npm от пользователя (без root)

June 6, 2017

1 minute read

  • Twitter

  • Twitter

  • Google+

  • Facebook

  • LinkedIn

  • Email

При глобальной установке node/npm через менеджер пакетов есть проблема: практически 100% поставится старая версия, с которой ничего уже не работает.

Поэтому мы будем ставить свежую версию nodejs локально от пользователя (без root).

Подсказки по локальной установке: isaacs/node-and-npm-in-30-seconds.sh. В инструкции ниже используется вариант с nave.

Подготовка: basher и nave

basher нужен для установки nave

Basher на github: https://github.com/basherpm/basher.

git clone https://github.com/basherpm/basher.git ~/.basher
echo 'export PATH="$HOME/.basher/bin:$PATH"' >> ~/.bash_profile
echo 'eval "$(basher init -)"' >> ~/.bash_profile

Обновляем окружение:

. ~/.bash_profile

nave нужен для установки nodejs

Установка nave:

basher install isaacs/nave

Установка nodejs

Маленький хак для локальной установки

Сначала делаем хак, чтобы он поставился локально, а не глобально:

mkdir -p ~/.local/bin
touch ~/.local/bin/node
chmod +x ~/.local/bin/node
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bash_profile

Обновляем env

. ~/.bash_profile

Собственно установка nodejs

Это поставит последнюю стабильную версию node:

nave usemain stable

Установка npm

npm уже поставлен вместе с node, но лучше его сразу обновить:

npm install npm -g

Что дальше?

Теперь установка любого пакета происходит элементарно, например:

npm install maildev -g
